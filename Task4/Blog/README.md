# Blog 项目

## 中文说明

### 运行环境要求
- Go 1.21+
- MySQL 5.7+

### 依赖安装
```bash
go mod download
```

### 启动方式
```bash
go run main.go
```

### 数据库配置
请确保MySQL服务已启动，并在config/DbConfig.go中修改数据库连接信息

## English Version

### Requirements
- Go 1.21+
- MySQL 5.7+

### Install Dependencies
```bash
go mod download
```

### Run Application
```bash
go run main.go
```

### Database Configuration
Make sure MySQL service is running and modify database connection in config/DbConfig.go
